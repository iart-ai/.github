## Install

Each pack installs on its own:

```
npx skills add iart-ai/short-form-video-skills
npx skills add iart-ai/data-video-skills
# …or any pack above
```

Skills are auto-discovered by Claude Code and 40+ other agents — once installed, they activate when your prompt matches the work.

## What's a skill?

A skill is a self-contained folder (`SKILL.md` + `references/`) that teaches an agent a specific motion-graphics workflow — kinetic typography, beat-synced cuts, accurate data charts, accessible web animation, and so on. Browse any pack to see what's inside.
